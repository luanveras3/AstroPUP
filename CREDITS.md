## Original foundation

AstroPUP includes code derived from:

- PUPRemote by Anton's Mindstorms
- Original authors: Anton Vanhoucke & Ste7an
- Original project: https://github.com/antonvh/PUPRemote

AstroPUP keeps attribution to the original project and adds:

- safer hub-side calls;
- safer sensor-side processing;
- startup reports;
- remote mode validation;
- counters;
- optional heartbeat / stale-data detection;
- clearer generic structure for robotics education and competitions.
